# Swinglio

Swinglio is a mobile app for tracking your golf rounds. Score holes in real time, keep detailed stats, and maintain a WHS-compliant handicap index -- all from your phone.

## What It Does

- **Live scoring** -- Record scores hole-by-hole during a round, with stat tracking for fairways hit, putts, greens in regulation, bunkers, and penalties.
- **Handicap tracking** -- Automatically calculates and maintains your handicap index following the World Handicap System.
- **Play with friends** -- Add friends, start rounds together, and view each other's scorecards in real time.
- **Course database** -- Search from 30,000+ US golf courses with hole-by-hole yardage and par data.
- **Course maps** -- View hole layouts with distance markers and green center locations powered by Mapbox.
- **Round history** -- Review past rounds, see scoring trends, and track your progress over time.
- **Attestation** -- Sign off on each other's rounds for official handicap tracking.

## Built With

React Native (Expo), Supabase, Mapbox, and TypeScript. See the [docs](./docs/) folder for technical details.
